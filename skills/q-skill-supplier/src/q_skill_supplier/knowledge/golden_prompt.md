# 供应商质量分析 - 黄金 Prompt

## 分析目标

对供应商来料质量进行全面评估，识别质量风险，提出改进建议。

## 核心指标

1. **合格率** = (检验数 - 不良数) / 检验数 × 100%
2. **PPM** = 不良数 / 检验数 × 1,000,000
3. **趋势** = 近期合格率与历史对比

## 预警阈值

- 🟢 优秀：合格率 ≥ 99%
- 🟡 正常：合格率 95% ~ 99%
- 🔴 预警：合格率 < 95%

## 分析逻辑

### 第一步：数据概览
- 统计检验总量、不良总量
- 计算综合合格率

### 第二步：供应商排名
- 按合格率从低到高排序
- 识别预警供应商

### 第三步：趋势分析
- 对预警供应商分析近期趋势
- 判断是持续恶化还是偶发问题

### 第四步：根因初判
- 分析主要不良类型
- 关联供应商历史表现

### 第五步：改进建议
- 针对预警供应商提出具体措施
- 考虑约谈、加严检验、辅导改善等手段

## 报告格式

1. 总体概况（数据汇总）
2. 供应商排名表
3. 预警供应商清单
4. 趋势分析结论
5. 改进建议

## 经验积累

每次分析完成后，应保存以下经验：
- 供应商名称
- 主要问题描述
- 采取的措施
- 效果追踪点
