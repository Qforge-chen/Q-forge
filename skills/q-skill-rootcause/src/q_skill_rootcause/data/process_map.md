# PM（粉末冶金）铁基齿轮 压制-烧结-整形 工艺解析
 
## 1. 工艺流程概览 (Process Flow)
 
> **Powder Prep & Blending** -> **Compaction (Die Pressing)** -> **Ejection & Green Handling** -> **Delubing/Debinding** -> **Sintering** -> **Sizing/Coining** -> **(Optional) Machining** -> **Inspection**
 
---
 
## 2. 详细工序分析
 
### 2.1 压制成型 (Compaction / Pressing)
*   **动作**: 将混合好的粉末装填到模腔，进行模压形成绿坯 (Green Compact)。
*   **关键机制**: 颗粒重排 + 变形致密化；密度分布与应力分布由装粉、压制与顶出共同决定。
*   **KPIV (关键输入变量)**:
    *   **装粉一致性**: 充填不足/偏析会导致密度梯度。
    *   **压制压力/速度/保压**: 过快或排气不足易出现层裂 (Lamination)。
    *   **模具通气与润滑**: 影响排气与顶出摩擦。
    *   **顶出力 (Ejection force)**: 过高常指向摩擦异常、润滑偏差或模具问题。
*   **潜在缺陷**: 层裂、绿坯微裂纹、密度不均、掉边。
 
### 2.2 脱脂/去润滑剂 (Delubing / Debinding)
*   **动作**: 去除润滑剂/粘结剂（加热或气氛处理），为烧结致密化做准备。
*   **关键机制**: 挥发/分解产物需要顺利排出；排出受阻会形成孔隙带或微裂纹起源。
*   **KPIV**:
    *   **升温速率与保温段**: 过快会导致气体滞留。
    *   **排气/流量/炉内装载**: 装载过密会影响气体通道。
 
### 2.3 烧结 (Sintering)
*   **动作**: 在设定温度窗口与气氛中使颗粒间形成烧结颈并致密化。
*   **关键机制**: 烧结颈生长 + 孔隙收缩；温度/时间/气氛与均匀性决定强度与孔隙率。
*   **KPIV**:
    *   **峰值温度 & 保温时间**: 偏低可能导致烧结颈不足，偏高可能引起组织异常。
    *   **炉温均匀性/带速/装炉位置**: 均匀性不足会导致批内性能差异。
    *   **气氛控制**: 影响氧化/脱碳/表面状态（记录缺失时只能标注 Not provided）。
 
### 2.4 整形/定径 (Sizing / Coining)
*   **动作**: 对烧结件施加载荷进行尺寸修正与表面整形。
*   **关键机制**: 整形常是“触发窗口”：如果材料强度不足或内部存在缺陷，整形载荷会把潜在缺陷放大为可见裂纹。
*   **KPIV**:
    *   **整形力/位移曲线**: 异常峰值/波动可能对应工装、对中或润滑问题。
    *   **工装磨损与同轴度**: 易造成局部应力集中（齿根/轮毂过渡区）。
 
### 2.5 （可选）机加工 (Machining)
*   **动作**: 对关键尺寸进行切削或去毛刺。
*   **关键机制**: 切削力/热影响可使已存在的微缺陷扩展；但通常不应被默认当成“唯一根因”。

## 3. 核心质量特性 (KPOVs)
 
| 质量特性 | 常见缺陷 | 最可能的物理来源 (Mechanism) |
| :--- | :--- | :--- |
| **密度/孔隙率** | 孔隙偏高/孔隙带 | 1. 脱脂排气不畅<br>2. 烧结窗口偏离 |
| **齿轮开裂 (Cracking)** | 整形后裂纹显性化 | 1. 压制/顶出层裂或微裂纹<br>2. 脱脂缺陷导致孔隙带<br>3. 烧结颈不足/均匀性差导致强度不足<br>4. 整形工装/载荷触发应力集中<br>5. 转运磕碰/人为损坏 |
| **尺寸精度** | 齿形/内孔超差 | 1. 整形窗口不稳定<br>2. 工装磨损或对中问题 |
 
## 4. 侦探排查逻辑 (Detective Logic)
 
*   **若裂纹在整形后显性化**：先确认整形载荷/工装是否异常（触发窗口），同时并行评估“材料是否变脆/强度不足”。
*   **四大工艺归因（不预设）**：压制/顶出、脱脂、烧结、整形/加工触发。
*   **第五归因（现场线索）**：如果四大工艺证据均不支持或信息不足，必须建议去现场排查转运磕碰、包装、防护、混料混批与人为损坏。
